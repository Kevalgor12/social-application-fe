## Overview

Frontend for MERN Stack practical assignment built using React.

## Tech Stack

- React.js
- TypeScript
- TanStack Query (React Query)
- Formik
- Yup
- SCSS
- Fetch

## Features

- User Login & Registration
- User's post list with search and pagination
- Create, view, like and unlike posts
- Logged-in user profile

## State & API Handling

- TanStack Query is used for API calls
- useQuery for fetching data
- useMutation for create/update actions

## Forms & Validation

- Formik for form handling
- Yup for validation
- Proper email format required
- Password minimum length enforced
